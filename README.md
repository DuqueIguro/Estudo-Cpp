## Descrição

Este repositório é dedicado ao estudo e aprimoramento de habilidades em **C++**. Aqui, você encontrará exemplos práticos, exercícios e projetos criados com o objetivo de explorar os recursos da linguagem e consolidar conhecimentos.

## Objetivos

- Aprender e praticar conceitos básicos e avançados de C++.
- Desenvolver habilidades em programação orientada a objetos (POO).
- Explorar estruturas de dados e algoritmos.
- Criar pequenos projetos e programas para aplicar os conhecimentos adquiridos.

## Pré-requisitos

Antes de explorar o conteúdo deste repositório, certifique-se de ter o seguinte instalado em sua máquina:

- Compilador C++ (como GCC ou Devc++)
- CMake (opcional, para projetos mais avançados)
- Editor ou IDE de sua preferência (como Visual Studio Code, Devc++ ou outros)

## Como Usar

1. Clone este repositório para sua máquina local:
   ```bash
   git clone https://github.com/DuqueIguro/Estudo-Cpp.git
   ```

2. Navegue até o diretório do projeto:
   ```bash
   cd Estudo-Cpp
   ```

3. Compile e execute os arquivos desejados. Por exemplo:
   ```bash
   g++ src/exemplo.cpp -o exemplo
   ./exemplo
   ```

4. Explore os códigos e modifique-os para consolidar seu aprendizado.
